# 🤖 AI React Tool (Gemini API Powered)

An AI-powered web application built using **React + Vite** that allows users to ask questions and get intelligent responses using Google's Gemini API.

---

## 🚀 Features

* 💬 Ask anything (AI chatbot style)
* ⚡ Fast responses using Gemini API
* 🎨 Clean and responsive UI
* 🔥 Built with modern React (Hooks)
* 🌐 API integration with Google Generative AI

---

## 🛠️ Tech Stack

* **Frontend:** React, Vite, JavaScript
* **Styling:** CSS / Tailwind CSS
* **API:** Google Gemini API

---



## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/AI_REACT_TOOL.git
cd AI_REACT_TOOL
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the project

```bash
npm run dev
```
